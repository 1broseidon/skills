Principles to hold

  • Prove by running. A verb that has never been exercised against a real artifact is a hypothesis, not a feature. The derivation gate caught real drift twice because we ran it.
  • Advisory before tooling. No build step means consistency checks, not generators. Add tooling only when a site is large enough that hand-syncing hurts.
  • One canonical manifest. Stamps and the index derive from site.json. Triple-write is a drift factory; derivation is the cure.
