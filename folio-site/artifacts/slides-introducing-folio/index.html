Folio

Static artifacts that open in a browser

The problem

  • Agents fabricate backends that do not exist.
  • They paste mystery CDNs into otherwise local artifacts.
  • They claim fake publish URLs without command evidence.

The discipline

  • Static documents only: no live network calls or socket streams.
  • Manifest ledger before build: deps, data, mocks, URLs.
  • Local-first: a folder on disk before optional publish.
