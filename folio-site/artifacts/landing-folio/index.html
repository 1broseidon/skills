Folio

Static artifacts that open in a browser.

Folio helps local agents create honest, self-contained HTML/CSS/JS artifacts for docs, data, demos, tools, and handoff.

Artifact kinds

doc

A readable document with structure, tables, and print-friendly prose.

sheet

Tabular data to scan, sort, filter, and summarize.

dashboard

Metrics at a glance with static charts and status gates.

explainer

A step-by-step walkthrough or data story with small illustrations.

slides

A linear presentation with keyboard and touch navigation.

tool

One local task completed with vanilla controls and copyable output.

landing

Orientation, navigation, and concise rules for a public visitor.

The rules

Static-only. No live network calls, socket streams, or XHR. Data is embedded, generated, or loaded from local files.

Manifest honesty. Dependencies, mock data, local files, and publish targets are recorded before build and checked at handoff.

Local-first. A runnable folder on disk comes before optional publish; URLs are only reported from real command output.
